HashSet - класс, реализует интерфейс Set и наследуется от класса AbstractSet
HashSet является одной из фундаментальных структур данных в API коллекций Java.

Хэш-функция — это функция, сужающая множество значений объекта до некоторого подмножества целых чисел.

 - Он хранит уникальные элементы и разрешает нули
 - Он поддерживается HashMap
 - Не поддерживает порядок вставки
 - Это не потокобезопасный

------------------------------------------------------------------------------------------------------------------------

Конструкторы HashSet:

  // Создание пустого набора с начальной емкостью (16) и со значением коэффициента загрузки (0.75) по умолчанию
  public HashSet();

  // Создание множества из элементов коллекции
  public HashSet(Collection c);

  // Создание множества с указанной начальной емкостью и со значением коэффициента загрузки по умолчанию (0.75)
  public HashSet(int initialCapacity);

  // Создание множества с указанными начальной емкостью и коэффициентом загрузки
  public HashSet(int initialCapacity, float loadFactor);

  HashSet<Integer> myHashSet = new HashSet<>();

------------------------------------------------------------------------------------------------------------------------

Реализация HashSet не синхронизируется!!!
Если многократные потоки получают доступ к набору хеша одновременно, а один или несколько потоков должны изменять набор,
то он должен быть синхронизирован внешне.
Это лучше всего выполнить во время создания, чтобы предотвратить случайный несинхронизируемый доступ к набору:

Set<E> set = Collections.synchronizedSet(new HashSet<E>());
