Set(Множества) - интерфейс. Расширяет интерфейс Collection.
Реализация интерфейса Set представляет собой неупорядоченную коллекцию объектов, которая не содержит дубли.

К семейству интерфейса Set относятся:
 - HashSet,
 - TreeSet,
 - LinkedHashSet.

В множествах Set разные реализации используют разный порядок хранения элементов.
 - В HashSet порядок элементов оптимизирован для быстрого поиска.
 - В контейнере TreeSet объекты хранятся отсортированными по возрастанию.
 - LinkedHashSet хранит элементы в порядке добавления.

------------------------------------------------------------------------------------------------------------------------

 - Метод add(Object o) - добавление элемента в коллекцию, если он отсутствует. Возвращает true, если элемент добавлен.
   set.add(1);

 - Метод addAll(Collection c) - добавление элементов коллекции, если они отсутствуют.

 - Метод clear() - очистка коллекции.
   set.clear();

 - Метод contains(Object o) - проверка присутствия элемента в наборе. Возвращает true, если элемент найден.
   set.contains(1);


 - Метод containsAll(Collection c) - проверка присутсвия коллекции в наборе. Возвращает true, если все элементы содержатся в наборе.

 - Метод equals(Object o) - проверка на равенство.

 - Метод hashCode() - получение hashCode набора.

 - Метод isEmpty() - проверка наличия элементов. Возвращает true если в коллекции нет ни одного элемента.
   set.isEmpty();

 - Метод iterator() - функция получения итератора коллекции.

 - Метод remove(Object o) - удаление элемента из набора.
   set.remove(1);

 - Метод removeAll(Collection c) - удаление из набора всех элементов переданной коллекции.

 - Метод retainAll(Collection c) - удаление элементов, не принадлежащих переданной коллекции.

 - Метод size() - количество элементов коллекции.
   set.size();

 - Метод toArray() - преобразование набора в массив элементов.

 - Метод toArray(T[] a) - преобразование набора в массив элементов. В отличии от предыдущего метода,
   который возвращает массив объектов типа Object, данный метод возвращает массив объектов типа, переданного в параметре.

 - Методы перебора:
   for (int i : set) {
       System.out.println(i);
   }
